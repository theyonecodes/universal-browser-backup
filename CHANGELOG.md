# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-19

### Added
- Initial release
- Multi-browser support (Chrome, Edge, Firefox, Brave, Opera, Vivaldi, Thorium)
- Automatic browser detection
- Visual browser selection with colored icons
- Profile size preview
- Backup and restore functionality
- Manifest system for backup verification
- Safety features (browser check, profile backup)
- Auto-launch option after restore
- Comprehensive documentation (README, BRD, PRD)
- MIT License

### Supported Browsers
- Google Chrome
- Microsoft Edge
- Mozilla Firefox
- Brave
- Opera
- Vivaldi
- Thorium
